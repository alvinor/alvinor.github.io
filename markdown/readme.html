流程图

非通用 gitlab 采用 mermaid 扩展

示例

```mermaid

graph LR

A(养车快现)-->B(贷款买车的用户);

B --> C{是否还清车贷};

C --> | 已还清 | End((End));

C --> | 未还清|D{有无养车贷};

D-->|无|E{是否有过额度测评};

D-->|有|F{是否还清养车贷};

F-->|未还清|G(还款计划);

F-->|已还清|H(还款计划);

E-->|无|I(额度测评);

E-->|有|J(查看额度);

G-->K(还款);

I-->L(提交资料);

L-->M(新网信审); ```
